# Typing Checkmark with Keyboard
## Solution
Type `10003` or `10004` on **numpad** while pressing `Alt` key.

Alt + 10003 | ✓
------------|--
Alt + 10004 | ✔

> Note: It doesn't work everywhere.
> Try enabling `Numlock` if failed.
>
> (Works in Office products)
